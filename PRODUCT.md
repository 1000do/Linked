# Product

## Register

product

## Users
Instructors and Learners on a course marketplace platform. Instructors manage their courses and track earnings. Learners browse, purchase, and consume educational content.

## Product Purpose
A robust course marketplace connecting educators with learners. Features comprehensive dashboard tools for instructors to manage content, sales, and analytics, while providing learners with an intuitive course discovery and learning experience.

## Brand Personality
Professional, modern, trustworthy, empowering.

## Anti-references
Cluttered corporate interfaces, generic Bootstrap templates without personality, dense data tables without visual hierarchy.

## Design Principles
- Clarity over decoration: Interfaces must surface key metrics immediately.
- Consistent visual rhythm: Use standard spacing and typography hierarchies.
- Reassuring precision: Financial and enrollment data should feel secure and exact.

## Accessibility & Inclusion
Ensure high contrast for text (especially data), clear focus states for interactive elements, and responsive layouts that work across devices.
