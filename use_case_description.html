2.2.2 Use Case Description
ID Use Case Actors Use Case Description
UC-01RegisterGuestAllows guests to register a new account using their email address, username, and password.
UC-02View Course ListGuest, User, InstructorDisplays a paginated list of published courses on the marketplace.
UC-03View Course DetailGuest, User, Instructor, Staff, AdminDisplays the detailed content of a course including chapters, lessons, instructor biography, ratings, and pricing.
UC-04Search CoursesGuest, User, InstructorSearches for courses based on titles, descriptions, or keywords.
UC-05Filter CoursesGuest, User, InstructorFilters courses by categories, price ranges, or minimum ratings.
UC-06Sort CoursesGuest, User, InstructorSorts the course lists by newest, price (lowest/highest), or ratings.

UC-12View purchased course listUser, InstructorDisplays all courses successfully purchased or enrolled in by the user.

UC-15Report A CourseUser, InstructorAllows enrolled users to report a course for quality issues or copyright violations.
UC-16Delete personal reviewsUser, InstructorAllows users to delete the course reviews they have previously submitted.
UC-17Gift CoursesUser, InstructorAllows users to purchase a course as a gift for another person instead of buying it for themselves.

UC-19Receive a giftUser, InstructorThis allows the recipient to receive the course from the giver and begin studying.

UC-44Search Transaction HistoryUser, InstructorSearches past transaction history by order ID or course name.
UC-45Request RefundUser, InstructorRequests a refund for a course transaction within the allowed policy time frame.

UC-47Forget PasswordUser, InstructorRequests a password reset verification link or OTP code via email.

UC-51View Chat HistoryUser, InstructorDisplays the chronological history of messages sent and received in a chat thread.
UC-52Search Chat HistoryUser, InstructorSearches past chat messages in a conversation by keyword.
