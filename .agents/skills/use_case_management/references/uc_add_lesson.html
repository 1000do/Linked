UC ID and Name UC-AddLesson - Add Lesson
Created By: System Date Created: 09/06/2026
Primary Actor: Instructor Secondary Actors: N/A
Trigger: Instructor clicks the "Create New Lesson" button or "Add Lesson" icon on the course syllabus.
Description: Allows an instructor to add a new lesson to an existing course. The system enforces business rules such as lesson limits, uniqueness of lesson titles, and restricts modifications if the course is currently pending review or the instructor is locked out.
Preconditions:

PRE-1. User must be successfully authenticated as an approved instructor.

PRE-2. The course must exist and belong to the active instructor.
Post-conditions:

POST-1. A new lesson is added to the course curriculum.

POST-2. If the course was previously published, its status is reverted to draft and moderation feedback is cleared.

POST-3. Stale cache data is cleared.
Normal Flow:

A. Add New Lesson Successfully

  1. Instructor accesses the course editor.
  2. Instructor clicks the "Create New Lesson" button or "Add Lesson" icon.
  3. The system displays the "Add New Lesson" modal.
  4. Instructor enters the lesson title in the "Lesson Title" input field.
  5. Instructor clicks the "Create Lesson" button.
  6. The system validates the instructor's rights.
  7. The system validates the course status.
  8. The system validates the maximum lesson limits.
  9. The system validates the lesson title for uniqueness.
  10. The system creates the new lesson, updates the course status to draft, and clears previous moderation feedback in the database.
  11. The system invalidates the cached course data.
  12. The system adds the newly created lesson to the curriculum list.
  13. The system closes the "Add New Lesson" modal.
Alternative Flows:

A.10 Course is not published

  1. The system skips updating the course status and moderation feedback.
  2. Back to step 11.
Exceptions:

EX-1.1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-1.2 Course Not Found

  1. The system displays a course not found error message.
  2. The system redirects the user back to the course list page.

EX-6 Invalid Instructor Rights

  1. The system detects the instructor is unauthorized to add a lesson to this course or has an active lockout status.
  2. The system displays an error message regarding invalid instructor rights.
  3. Back to step 3.

EX-7 Invalid Course Status

  1. The system detects the course is pending review or is permanently archived due to policy violations.
  2. The system displays an error message regarding invalid course status.
  3. Back to step 3.

EX-8 Max Lessons Reached

  1. The system detects the course already contains 5 lessons and the instructor has not linked an active Stripe Connect profile.
  2. The system displays an error message regarding maximum lesson limits.
  3. Back to step 3.

EX-9 Duplicate Lesson Title

  1. The system detects a lesson with the same title already exists in the course (case-insensitive).
  2. The system displays an error message regarding duplicate lesson titles.
  3. Back to step 3.

EX-10 Database Save Failed

  1. The system fails to save the lesson changes to the database.
  2. The system displays a database transaction failure message.
  3. Back to step 3.
Priority: Medium
Frequency of Use: High
Business Rules:
  1. Lesson Limit for Unlinked Stripe Accounts: Instructors who have not linked an active Stripe Connect profile are only allowed to create up to 5 lessons per course.
  2. Duplicate Title Check: Lesson titles must be unique within a course (case-insensitive).
  3. Pending Restriction: Lessons cannot be added to a course while it is currently in 'Pending' review status.
  4. Status Reversion: Adding a lesson to a 'Published' course automatically changes its status back to 'Draft' and clears any existing moderation feedback.
  5. Locked Out Instructor Rights: Instructors with an active lockout status are prohibited from creating lessons.
Other Information: N/A
Assumptions:

Instructor has stable internet connection.
