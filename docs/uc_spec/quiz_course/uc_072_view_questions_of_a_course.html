UC ID and Name UC-72-View Course Questions
Created By: Truong Phuoc Tai Date Created: 11/07/2026
Primary Actor: Instructor Secondary Actors: N/A
Trigger: Actor selects a lesson in the question bank page.
Description: Allows an Instructor to view a list of questions associated with a specific lesson for a course they own.
Preconditions:

PRE-1. Actor must be the owner of the course containing the lesson.
Post-conditions:

POST-1. The list of questions for the lesson is displayed to the actor.
Normal Flow:

A. View Questions Successfully

  1. Actor selects a lesson in the question bank page.
  2. The system retrieves the lesson details.
  3. The system validates the course ownership for the actor.
  4. The system retrieves the list of questions associated with the lesson.
  5. The system maps the questions to the response format.
  6. The system displays the list of questions to the actor.
Alternative Flows: N/A
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Lesson Not Found

  1. The system detects the requested lesson does not exist.
  2. The system displays a lesson not found error message.
  3. Back to step 1.

EX-3 Unauthorized Access

  1. The system detects the actor is not the owner of the course.
  2. The system displays an unauthorized error message.
  3. Back to step 1.
Priority: High
Frequency of Use: High
Business Rules: N/A
Other Information: N/A
Assumptions:

Instructor has stable internet connection.
