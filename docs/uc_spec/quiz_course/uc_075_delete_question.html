UC ID and Name UC-75-Delete Question
Created By: Truong Phuoc Tai Date Created: 11/07/2026
Primary Actor: Instructor Secondary Actors: N/A
Trigger: Actor confirms the deletion of a question.
Description: Allows an Instructor to permanently delete a quiz question from their question bank.
Preconditions:

PRE-1. Actor must be the owner of the course containing the question.
Post-conditions:

POST-1. The question is deleted from the database.

POST-2. The actor is presented with a success message.
Normal Flow:

A. Delete Question Successfully

  1. Actor confirms the deletion of a question.
  2. The system retrieves the question details.
  3. The system retrieves the course details associated with the question.
  4. The system validates the course ownership for the actor.
  5. The system removes the question entity.
  6. The system saves the deletion to the database.
  7. The system displays a success message that the question was deleted successfully.
Alternative Flows:

A.1 Cancel Deletion

  1. Actor clicks the cancel button on the deletion confirmation modal.
  2. The system closes the modal.
  3. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Question Not Found

  1. The system detects the requested question does not exist.
  2. The system displays a not found error message.
  3. Back to step 1.

EX-3 Course Not Found Or Unauthorized

  1. The system detects the course is not found or the actor is not the owner.
  2. The system displays an unauthorized error message.
  3. Back to step 1.

EX-6 Database Delete Failure

  1. The system encounters an error while deleting from the database.
  2. The system displays a server error message.
  3. Back to step 1.
Priority: High
Frequency of Use: Medium
Business Rules: N/A
Other Information: N/A
Assumptions:

Instructor has stable internet connection.
