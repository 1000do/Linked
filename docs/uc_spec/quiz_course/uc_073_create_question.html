UC ID and Name UC-73-Create Question
Created By: Truong Phuoc Tai Date Created: 11/07/2026
Primary Actor: Instructor Secondary Actors: N/A
Trigger: Actor submits the add question form.
Description: Allows an Instructor to create a new quiz question and its associated options for a course they own.
Preconditions:

PRE-1. Actor must be the owner of the course.
Post-conditions:

POST-1. The new question and its options are saved to the database.

POST-2. The actor is presented with a success message.
Normal Flow:

A. Create Question Successfully

  1. Actor submits the add question form.
  2. The system validates the submitted quiz options.
  3. The system retrieves the course details.
  4. The system validates the course ownership for the actor.
  5. The system creates a new question and its options entities.
  6. The system saves the new question and options to the database.
  7. The system maps the created question to the response format.
  8. The system displays a success message that the question was added successfully.
Alternative Flows:

A.1 Invalid Form Submission

  1. Actor attempts to submit the form with invalid or missing data.
  2. The system validates the form inputs.
  3. The system detects validation errors.
  4. The system displays frontend validation error messages.
  5. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Invalid Options

  1. The system detects the submitted quiz options are invalid.
  2. The system displays a validation error message.
  3. Back to step 1.

EX-3 Course Not Found Or Unauthorized

  1. The system detects the course is not found or the actor is not the owner.
  2. The system displays an unauthorized error message.
  3. Back to step 1.

EX-6 Database Save Failure

  1. The system encounters an error while saving to the database.
  2. The system displays a server error message.
  3. Back to step 1.
Priority: High
Frequency of Use: High
Business Rules: BR-279, BR-280, BR-281
Other Information: N/A
Assumptions:

Instructor has stable internet connection.
