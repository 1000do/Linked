UC ID and Name UC-15-Submit Quiz Attempt
Created By: Truong Phuoc Tai Date Created: 11/07/2026
Primary Actor: User Secondary Actors: N/A
Trigger: Actor clicks the "Submit" button after completing the quiz.
Description: Allows a User to submit their ongoing quiz attempt. The system calculates the score based on the submitted answers, updates the attempt record, and displays the result summary.
Preconditions:

PRE-1. Actor must have an active, unsubmitted quiz attempt.
Post-conditions:

POST-1. The quiz attempt is marked as submitted with a calculated score.

POST-2. The actor is presented with the quiz result summary.
Normal Flow:

A. Submit Quiz Attempt Successfully

  1. Actor clicks the "Submit" button on the quiz learning page.
  2. The system retrieves the quiz attempt details.
  3. The system validates that the actor is the owner of the attempt.
  4. The system validates that the attempt has not been submitted yet.
  5. The system retrieves the quiz details for scoring.
  6. The system calculates the score for each submitted answer.
  7. The system calculates the total score and the pass/fail status.
  8. The system updates the attempt record with the final score, submission time, and status.
  9. The system saves the updated attempt record to the database.
  10. The system displays the quiz result summary to the actor.
Alternative Flows:

A.1 Cancel Submission

  1. Actor clicks the cancel button on the submission confirmation modal.
  2. The system closes the modal.
  3. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Attempt Not Found

  1. The system detects the requested attempt does not exist.
  2. The system displays an attempt not found error message.
  3. Back to step 1.

EX-3 Unauthorized Access

  1. The system detects the actor is not the owner of the attempt.
  2. The system displays an unauthorized access error message.
  3. Back to step 1.

EX-4 Already Submitted

  1. The system detects the attempt has already been submitted.
  2. The system displays an error message stating the quiz was already submitted.
  3. Back to step 1.

EX-9 Database Save Failure

  1. The system encounters an error while saving to the database.
  2. The system displays a server error message.
  3. Back to step 1.
