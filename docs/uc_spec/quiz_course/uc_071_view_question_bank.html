UC ID and Name UC-71-View Question Bank
Created By: Truong Phuoc Tai Date Created: 11/07/2026
Primary Actor: Instructor Secondary Actors: N/A
Trigger: Actor selects a course in the question bank page.
Description: Allows an Instructor to view a summary of lessons and their associated question counts for a specific course they own in the question bank.
Preconditions:

PRE-1. Actor must be the owner of the course.
Post-conditions:

POST-1. The lessons summary for the course is displayed to the actor.
Normal Flow:

A. View Question Bank Successfully

  1. Actor selects a course in the question bank page.
  2. The system validates the course ownership for the actor.
  3. The system retrieves the list of lessons for the course.
  4. The system retrieves the questions associated with the course.
  5. The system maps the lessons and questions to the lesson summary response format.
  6. The system displays the lessons summary to the actor.
Alternative Flows: N/A
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Course Not Found Or Unauthorized

  1. The system detects the course is not found or the actor is not the owner.
  2. The system displays an unauthorized error message.
  3. Back to step 1.
Priority: High
Frequency of Use: High
Business Rules: N/A
Other Information: N/A
Assumptions:

Instructor has stable internet connection.
