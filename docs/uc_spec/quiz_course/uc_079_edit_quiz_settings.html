UC ID and Name UC-79-Edit Quiz Settings
Created By: Truong Phuoc Tai Date Created: 11/07/2026
Primary Actor: Instructor Secondary Actors: N/A
Trigger: Actor modifies quiz settings and clicks Save.
Description: Allows an Instructor to update the configurations and question distributions of an existing quiz they own.
Preconditions:

PRE-1. Actor must be the owner of the quiz.
Post-conditions:

POST-1. The updated quiz settings and distributions are saved to the database.

POST-2. The actor is presented with a success message.
Normal Flow:

A. Edit Quiz Settings Successfully

  1. Actor submits the updated quiz settings in the editor page.
  2. The system validates the quiz options and title uniqueness.
  3. The system validates the quiz question distributions.
  4. The system retrieves the existing quiz and its distributions from the database.
  5. The system validates the quiz ownership for the actor.
  6. The system updates the quiz properties and distributions.
  7. The system saves the changes to the database.
  8. The system maps the updated quiz to the response format.
  9. The system displays a success message that the quiz was updated successfully.
Alternative Flows:

A.1 Invalid Form Submission

  1. Actor attempts to submit the form with invalid or missing data.
  2. The system validates the form inputs.
  3. The system detects validation errors.
  4. The system displays frontend validation error messages.
  5. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Validation Failed

  1. The system detects the quiz options or title are invalid.
  2. The system displays a validation error message.
  3. Back to step 1.

EX-3 Distributions Invalid

  1. The system detects the configured question distributions are invalid.
  2. The system displays an invalid distribution error message.
  3. Back to step 1.

EX-4 Quiz Not Found

  1. The system detects the requested quiz does not exist.
  2. The system displays a quiz not found error message.
  3. Back to step 1.

EX-5 Unauthorized Access

  1. The system detects the actor is not the owner of the quiz.
  2. The system displays an unauthorized access error message.
  3. Back to step 1.

EX-7 Database Save Failure

  1. The system encounters an error while saving to the database.
  2. The system displays a server error message.
  3. Back to step 1.
