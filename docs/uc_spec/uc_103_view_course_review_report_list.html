UC ID and Name UC-103 - View List of Course Review Report
Created By: System Date Created: 09/06/2026
Primary Actor: Staff, Admin Secondary Actors: N/A
Trigger: Staff or Admin accesses the Report Center page and navigates to the "Review Reports" tab.
Description: Displays reported course reviews flagged by users.
Preconditions:

PRE-1. The actor must be authenticated.

PRE-2. The actor must have Staff or Admin role.
Post-conditions:

POST-1. The system displays the paginated list of pending course review reports.
Normal Flow:

A. View List of Course Review Report Successfully

  1. Staff or Admin accesses the Report Center page.
  2. The system validates the actor's authentication and authorization.
  3. The actor selects the "Review Reports" tab.
  4. The system displays the review subtabs, with "Course Reviews" selected by default.
  5. The system retrieves the paginated list of pending course review reports.
  6. The system displays the list of course review reports.
Alternative Flows:

A.5 Empty Course Review Report List

  1. The system detects the retrieved list of course review reports is empty.
  2. The system displays an empty state message indicating that no review reports were found.
  3. Use case ends.
Exceptions:

EX-2.1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2.2 Unauthorized Access

  1. The system detects the actor does not have the Staff or Admin role.
  2. The system redirects the actor to the home page or access denied page.

EX-5 Database Query Failure

  1. The system encounters an error while retrieving the reports from the database.
  2. The system displays an error message regarding the data retrieval failure.
  3. Back to step 4.
Priority: Medium
Frequency of Use: High
Business Rules: BR-10, BR-11, BR-12, BR-13, BR-14
Other Information: N/A
Assumptions:

Staff, Admin has stable internet connection.
