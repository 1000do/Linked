UC-75 - View Profile (For Manager)

Primary Actor:
Manager

Secondary Actors:
None

Trigger:
Manager accesses the profile page.

Description:
Allows managers (Super Admin or Staff Support) to view their profile details, including system account information.

Preconditions:

PRE-1. Manager must be successfully authenticated as an approved administrator or staff support account.

Post-conditions:

POST-1. The system displays the manager's profile, system account information, and status.

Normal Flow

A. View Profile Successfully

  1. Manager accesses the profile page.
  2. The system validates the login session and retrieves the manager's profile data.
  3. The system displays the manager profile details and system information (profile picture, verification status, name, email, biography, phone number, creation date, and suspension details if active).

Alternative Flows

None

Exceptions

EX-2 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-3 Manager Profile Not Found

  1. The system fails to retrieve the manager profile.
  2. The system displays an error message regarding profile retrieval failure.
  3. The system redirects the manager to the login page.

Business Rules

  1. Lockout Status Monitoring: The profile displays any active suspension records (start and end times) for the manager account, keeping administrative users informed of active lockout configurations.
  2. Role-based Labeling: The interface shows "Super Admin" if the manager's role is "ADMIN", and "Staff Support" otherwise.
  3. Information Display Isolation: The profile page is strictly isolated to the authenticated manager account's details.

Assumptions

Manager has stable internet connection.
