UC ID and Name UC-07-View Instructor Profile
Created By: Truong Phuoc Tai Date Created: 13/06/2026
Primary Actor: Guest, User, Instructor Secondary Actors: N/A
Trigger: Actor clicks on the instructor's name or profile link.
Description: Displays the public profile page of an instructor along with their list of created courses.
Preconditions:

PRE-1. The instructor's application must be approved.
Post-conditions:

POST-1. The instructor's public profile page is rendered, displaying their biography, social links, statistical information, and a list of published courses.
Normal Flow:

A. View Instructor Profile Successfully

  1. Actor clicks on an instructor's name or profile link.
  2. The system retrieves the instructor's ID from the URL request.
  3. The system retrieves the instructor's profile details (avatar, full name, professional title, expertise categories, biography, and social links).
  4. The system retrieves the instructor's statistical information (total students, average rating).
  5. The system counts the instructor's active courses.
  6. The system retrieves the instructor's published courses.
  7. The system displays the public profile page containing the instructor's avatar, full name, professional title, expertise categories, biography, social links ("LinkedIn", "YouTube", "Facebook"), and statistics (total students, active courses, average rating, total reviews).
  8. The system displays a list of course card items for each published course, showing the course thumbnail, title, rating, price, and enrollment count.
Alternative Flows:

A.8.1 View Course Details

  1. Actor clicks on a course card item.
  2. The system redirects the actor to the course details page.
Exceptions:

EX-3 Instructor Not Found or Not Approved

  1. The system detects that the instructor does not exist or has not been approved.
  2. The system redirects the actor to the home page.
Priority: High
Frequency of Use: Medium
Business Rules:
  1. Course Visibility: Only courses with status "Published" are displayed on the public profile. Draft, pending, rejected, or archived courses are strictly excluded.
  2. Profile Access: A public profile is only accessible if the instructor's account application status is "Approved".
  3. Social Links: Social media links ("LinkedIn", "YouTube", "Facebook") are only rendered as clickable buttons/links if the instructor has provided the respective URLs.
Other Information: N/A
Assumptions:

Guest has stable internet connection.

User has stable internet connection.

Instructor has stable internet connection.
