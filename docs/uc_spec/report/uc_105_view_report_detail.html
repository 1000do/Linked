UC ID and Name UC-105 - View Report Detail
Created By: System Date Created: 09/06/2026
Primary Actor: Staff, Admin Secondary Actors: N/A
Trigger: Staff or Admin clicks on a specific report row in the Report Center (applies to course reports, course review reports, and lesson review reports).
Description: Displays details and evidence of a reported violation.
Preconditions:

PRE-1. The actor must be authenticated.

PRE-2. The actor must have Staff or Admin role.
Post-conditions:

POST-1. The system displays a popup containing the detailed information of the selected report.
Normal Flow:

A. View Report Detail Successfully

  1. Staff or Admin accesses the Report Center page.
  2. The system validates the actor's authentication and authorization.
  3. Staff or Admin clicks on a report row (course report, course review report, or lesson review report) in the report list.
  4. The system retrieves the selected report details.
  5. The system determines the report status does not contain resolution details.
  6. The system displays a popup containing the report details (Reporter, Submitted At, Status, Reported Content, Reason, and Description).
Alternative Flows:

A.5 View Resolved/Rejected Report Detail

  1. The system determines the report status contains resolution details (e.g., resolved, rejected).
  2. The system appends the Resolution Details (Moderator, Resolved At, Resolution Note) to the report details.
  3. Back to step 6.
Exceptions:

EX-2.1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2.2 Unauthorized Access

  1. The system detects the actor does not have the Staff or Admin role.
  2. The system redirects the actor to the home page or access denied page.

EX-4 Database Query Failure

  1. The system encounters an error while retrieving the report details from the database.
  2. The system displays an error message regarding the data retrieval failure.
  3. Use case ends.
Priority: Medium
Frequency of Use: High
Business Rules: BR-15, BR-16
Other Information: N/A
Assumptions:

Staff, Admin has stable internet connection.
