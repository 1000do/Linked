UC ID and Name UC-90-Sort Courses
Created By: Truong Phuoc Tai Date Created: 13/06/2026
Primary Actor: Staff, Admin Secondary Actors: N/A
Trigger: Actor selects a sorting option from the "sortBy" dropdown on the Course Moderation Page.
Description: Sorts courses in the moderation list by submission date or threat level.
Preconditions:

PRE-1. Actor must be authenticated.

PRE-2. Actor must have staff or admin role.

PRE-3. Actor must be on the Course Moderation Page.
Post-conditions:

POST-1. The system displays the course moderation list sorted by the selected criteria.
Normal Flow:

A. Sort Courses Successfully

  1. Actor selects a sorting option from the "sortBy" dropdown.
  2. The system retrieves the courses list sorted according to the selected option and other active filters.
  3. The system updates the course moderation table and statistics cards.
  4. The system displays the sorted course list.
Alternative Flows:

A.1.1 Switch Sorting Option

  1. Actor selects a different sorting option from the "sortBy" dropdown.
  2. The system retrieves the courses list sorted by the newly selected option.
  3. The system updates the course moderation table and statistics cards.
  4. The system displays the updated sorted course list.
  5. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the actor to the login page.

EX-2 Unauthorized Access

  1. The system detects that the actor does not have staff or admin roles.
  2. The system displays an access denied page or error message.
  3. The flow terminates.

EX-3 API Call Failed

  1. The system fails to retrieve the sorted course list or stats from the API.
  2. The system displays an error message regarding sorting failure.
  3. The system renders the Course Moderation page with an empty list.
  4. The flow terminates.
Priority: High
Frequency of Use: High
Business Rules: BR-120, BR-127, BR-125
Other Information: N/A
Assumptions:

Staff has stable internet connection.

Admin has stable internet connection.
