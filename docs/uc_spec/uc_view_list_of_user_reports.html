UC ID and Name UC-102 - View List of User Reports
Created By: Huynh Kien An Date Created: 15/07/2026
Primary Actor: Staff, Admin Secondary Actors: N/A
Trigger: Actor navigates to the Report Center page.
Description: Displays reported courses, reported course reviews, and reported lesson reviews submitted by users. All category reports are accessible via tabs and subtabs; Actor can switch between category subtabs without a page reload.
Preconditions:

PRE-1. The actor must be authenticated.

PRE-2. The actor must have Staff or Admin role.
Post-conditions:

POST-1. The system displays the paginated list of reports for the selected category.
Normal Flow:

A. View List of User Reports Successfully

  1. Actor accesses the Report Center page.
  2. The system validates the actor's authentication and authorization.
  3. The system retrieves the overall report statistics and the paginated lists of pending course reports, course review reports, and lesson review reports.
  4. The system displays the overall report statistics and main navigation tabs.
  5. The system displays the list of pending course reports by default.
  6. Actor views the list of user reports.
Alternative Flows:

A.3 Empty Report List

  1. The system detects no reports exist for one or more report categories.
  2. The system displays an empty state message in the affected category's report list.
  3. Back to step 6.

A.6.1 Actor Switches Category Subtab

  1. Actor clicks a different category tab or subtab.
  2. The system displays the report list for the selected category.
  3. Back to step 6.

A.6.2 Actor Navigates to Another Page

  1. Actor clicks a pagination button.
  2. The system reloads the page with the requested page of reports for the active category.
  3. Back to step 1.
Exceptions:

EX-2.1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2.2 Unauthorized Access

  1. The system detects the actor does not have the Staff or Admin role.
  2. The system redirects the actor to the home page or access denied page.

EX-3 Database Query Failure

  1. The system encounters an error while retrieving the reports from the database.
  2. The system displays an error message regarding the data retrieval failure.
  3. Back to step 1.
Priority: Medium
Frequency of Use: High
Business Rules: BR-9, BR-10, BR-11, BR-12, BR-13, BR-14
Other Information: N/A
Assumptions:

Staff, Admin has stable internet connection.
