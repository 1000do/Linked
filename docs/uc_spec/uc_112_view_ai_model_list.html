UC ID and Name UC-112 - View AI Model List
Created By: An Date Created: 11/06/2026
Primary Actor: Admin Secondary Actors: N/A
Trigger: Admin accesses the AI Service management page.
Description: Displays a paginated list of all registered AI moderation models, showing each model's name, version, type, process type, status, and last modified date. Admin can navigate between pages using pagination controls.
Preconditions:

PRE-1. Admin must be successfully authenticated.
Post-conditions:

POST-1. The AI model list is displayed to the Admin in paginated form.

POST-2. The model list is ordered by last modified date in descending order.
Normal Flow:

A. View AI Model List Successfully

  1. Admin accesses the AI Service management page.
  2. The system validates the Admin's access authorization.
  3. The system retrieves the paginated list of AI models from the database.
  4. The system displays the AI model list under the model management tab.
  5. Admin views the AI model list.
Alternative Flows:

A.3 No AI Models Registered

  1. The system detects no AI models are registered in the database.
  2. The system displays an empty state message in the model list area.
  3. Back to step 5.

A.5 Admin Navigates to Another Page

  1. Admin clicks the "Next" or "Previous" pagination button.
  2. The system reloads the page with the requested page of AI models.
  3. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Unauthorized Access

  1. The system detects the current user does not have Admin authorization.
  2. The system displays an unauthorized access error message.
  3. The system redirects the user to the login page.

EX-3 Database Query Failed

  1. The system fails to retrieve the AI model list from the database.
  2. The system displays an error message regarding the retrieval failure.
  3. Back to step 1.
Priority: Medium
Frequency of Use: Medium
Business Rules: BR-19, BR-25, BR-26, BR-28
Other Information: N/A
Assumptions:

Admin has stable internet connection.
