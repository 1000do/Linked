UC ID and Name UC-114 - View Activity Log
Created By: An Date Created: 11/06/2026
Primary Actor: Admin Secondary Actors: N/A
Trigger: Admin clicks the "AI Activity Logs" tab on the AI Service management page.
Description: Displays chronological paginated logs of AI moderation events across three categories: course moderation, course review moderation, and lesson review moderation. All category logs are loaded at page load; Admin can switch between category subtabs without a page reload.
Preconditions:

PRE-1. Admin must be successfully authenticated.
Post-conditions:

POST-1. The activity log list for the selected category is displayed to the Admin.
Normal Flow:

A. View Activity Log Successfully

  1. Admin accesses the AI Service management page.
  2. The system validates the Admin's access authorization.
  3. The system retrieves the paginated activity logs for all three moderation categories.
  4. Admin clicks the "AI Activity Logs" tab.
  5. The system displays the activity log list under the course moderation subtab by default.
  6. Admin views the activity log entries.
Alternative Flows:

A.3 No Logs Found

  1. The system detects no activity logs exist for one or more moderation categories.
  2. The system displays an empty state message in the affected category's log list.
  3. Back to step 6.

A.5 Admin Switches Category Subtab

  1. Admin clicks a different category subtab.
  2. The system displays the activity log list for the selected category.
  3. Back to step 6.

A.6 Admin Navigates to Another Page

  1. Admin clicks the "Next" or "Previous" pagination button.
  2. The system reloads the page with the requested page of logs for the active category.
  3. Back to step 1.
Exceptions:

EX-1 Session Invalid (Not Logged In)

  1. The system displays an authentication error message.
  2. The system redirects the user to the login page.

EX-2 Unauthorized Access

  1. The system detects the current user does not have Admin authorization.
  2. The system displays an unauthorized access error message.
  3. The system redirects the user to the login page.

EX-3 Database Query Failed

  1. The system fails to retrieve activity logs from the database.
  2. The system displays an error message regarding the retrieval failure.
  3. Back to step 1.
Priority: Medium
Frequency of Use: Medium
Business Rules: BR-19, BR-25, BR-26, BR-28, BR-29
Other Information: N/A
Assumptions:

Admin has stable internet connection.
