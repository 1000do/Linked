146 Search Courses Admin Course Searches course list by title or instructor name for moderation purposes. Actor must be authenticated and have staff or admin role. Actor must be on the Course Moderation Page.
147 Sort Courses Admin Course Sorts courses in the moderation list by submission date or threat level. Actor must be authenticated and have staff or admin role. Actor must be on the Course Moderation Page.
148 Filter Courses Admin Course Filters courses by categories or moderation status (Pending, Approved, Rejected). Actor must be authenticated and have staff or admin role. Actor must be on the Course Moderation Page.
149 Approve Course Submission Admin Course Approves a submitted course, marking it as published and visible on the marketplace. Actor must be authenticated and have staff or admin role. The course status must be pending review (or rejected, archived, permanently locked if re-approving).
150 Reject Course Submission Admin Course Rejects a submitted course with reasons and feedback for the instructor. Actor must be authenticated and have staff or admin role. The course status must be pending review (or approved/published if rejecting an active course).
151 Flag Course for Content Violation Admin Course Flags a course for content or policy violations, hiding it from public display. Actor must be authenticated and have staff or admin role. The course status must be pending, approved/published, rejected, or archived. The course's current flag count must be less than 3.
152 Publish courses Publish courses Allows instructors to request system approval to publish draft courses. The system validates the submission immediately, returns a confirmation status, and triggers AI-assisted moderation asynchronously. User must be successfully authenticated as an approved instructor. The course must exist and belong to the active instructor.
153 Add wishlist items Manage Wishlist Allows an authenticated user to add a course to their wishlist to save it for later. User must be logged in. The target course must exist and not already be in the wishlist.
154 View Wishlist Manage Wishlist Displays the list of courses in the user's wishlist. User must be logged in.
155 Delete Wishlist Items Manage Wishlist Allows an authenticated user to remove a course from their wishlist. User must be logged in. The course must exist in the user's wishlist.
156 Assign Quiz To Course Quiz Course Allows an Instructor to link an existing quiz to one of their courses within the course editor page. Actor must be the owner of the course.
