Checkout
TC-001 Process Checkout Successfully 1. Login as a User.
2. Add a course to the cart and navigate to checkout.
3. Review the cart and click "Checkout".
4. Enter valid Stripe payment card information on the Stripe secure form.
5. Click the checkout button on Stripe form.		 The system validates the transaction and records the order. The student is enrolled. The cart is cleared. The system calculates the instructor payout by deducting Stripe fees and system commission from the selling price (BR-282), schedules the payout in a "Pending" status for 14 days (BR-283), dynamically sets the currency based on the instructor's Stripe account country (BR-284), and displays a success notification. User is successfully authenticated. Cart contains at least one course. Passed 2026-07-19 HungPH Passed 2026-07-26 HungPH Passed 2026-08-02 HungPH
