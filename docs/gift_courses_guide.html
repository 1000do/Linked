3.19.1 Gift Courses

The purpose of this workflow is to allow the User to configure gift details for a course and proceed to checkout.

Execution steps:

Step 1: The user accesses the Course Details page and clicks the "Gift this course" button.

Step 2: The system displays the gift setup form. The user inputs information / selects actions:

  • Recipient's Email: Required. The email address of the person who will receive the gift. The system will verify if this user is already enrolled in the course.
  • Recipient's Name: Optional. The name of the person receiving the gift to personalize the gift card preview.
  • Choose Card Theme: Required. The visual theme selected for the gift card.
  • Gift Message: Optional. A personal message to include with the gift.
