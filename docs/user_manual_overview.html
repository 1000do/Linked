3. User Manual
3.1 Terms and definitions
No. Term and definition Description
1 Admin The system administrator who manages user accounts, system configurations, AI moderation settings, financial policies, and oversees the entire platform.
2 Staff A manager responsible for moderating courses, processing instructor applications, resolving user reports, and handling customer support chats.
3 Instructor A user who creates and publishes courses, manages learning materials and quizzes, and tracks their course revenues and payouts.
4 User A registered learner who can browse, purchase, and enroll in courses, take quizzes, and track their learning progress.
5 Guest An unregistered visitor who can browse published courses and view instructor profiles before registering an account.
3.2 System requirements
Web client:
  • Operating system: Windows 7, or newer.
  • Browser: Google Chrome, Edge, Brave, Opera, Firefox, etc.
