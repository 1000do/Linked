4.20.1. Start Quiz Attempt

Function Trigger: The user clicks the "Start Quiz" button on the quiz learning page.

Function Description: Begins taking a multiple-choice quiz.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must be logged in and enrolled in the course.
    • The quiz must be available (e.g., active, within allowable attempt limits).
  • Process
    • Start Quiz Attempt Successfully:
      • User accesses the quiz learning page.
      • User clicks the "Start Quiz" button.
      • The system validates the quiz availability and the user's enrollment status.
      • The system randomly selects questions based on the question distribution settings.
      • The system creates a new quiz attempt record.
      • The system displays the quiz questions for the user to answer.
  • Postconditions
    • A new quiz attempt record is saved to the database.
    • The user is presented with the quiz questions interface.
  • Error Handling
    • Session Invalid (Not Logged In): The system redirects the user to the Login page.
    • Quiz Not Found: The system displays a quiz not found error message.
    • Unauthorized Access or Quiz Unavailable: The system displays an error message regarding unauthorized access.
  • Security Measures
    • Attempt Limits Check: The system verifies if the user has reached the maximum number of allowed attempts before starting a new one.
