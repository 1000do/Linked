2.2.3.49. Setup payout

Function Trigger: Instructors click the "Setup Payout" or "Payout Settings" button in their instructor dashboard workspace.

Function Description: Allows approved instructors to link their financial profiles with Stripe Connect or enter manual banking credentials to receive earned course purchase commissions.

Function Details

  • Preconditions
    • - User must be successfully authenticated (logged in).
    • - The user's account must be approved as an Instructor by the system.
  • Process
    • Link Payout with Stripe Connect Successfully:
      • 1. Instructor accesses the Application Status page.
      • 2. The system retrieves the instructor's application status from the database.
      • 3. The system displays the approved application status along with the "Setup Payout" button.
      • 4. Instructor clicks the "Setup Payout" button.
      • 5. The system redirects the instructor to the Stripe onboarding page.
      • 6. On the Stripe onboarding page, the instructor completes the verification and banking setup: The instructor enters their contact details and inputs the OTP verification code; Instructor enters basic personal details; Instructor enters receiving bank details; Instructor reviews details and clicks the "Agree and submit" button.
      • 7. Stripe redirects the instructor back to the system's platform.
      • 8. The system displays a success message indicating that Stripe setup is completed.
      • 9. The system updates the instructor's payout status to active in the database.
      • 10. The system displays the active payout status along with the "Access Stripe Dashboard" button.
  • Alternative Flow (User Cancels Onboarding Mid-process)
    • 1. The user cancels the onboarding process on the Stripe page.
    • 2. The system redirects the instructor back to the platform.
    • 3. The system displays a warning message indicating that Stripe setup is not completed.
    • 4. The system displays the inactive payout status along with the "Setup Payout" button.
    • 5. Back to step 1.
  • Postconditions
    • - The instructor's bank account is successfully linked, and the payout status is displayed as "Active".
  • Error Handling
    • - Session Invalid (Not Logged In): Display the message: "Please login to view payout settings". Redirect to the login page.
    • - Stripe Onboarding Connection Failed: The system detects a connection failure with the payment gateway. The system displays an error message regarding connection failure. Back to step 3.
  • Security Measures
    • - Direct Edit Lockout: Once the bank account is successfully linked and status is "Active", the system hides the "Setup Payout" button. Instructors are locked out from editing bank details directly on the platform and must click the "Access Stripe Dashboard" button to edit directly on Stripe's secure portal.
