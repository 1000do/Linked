4.20.5. Assign Quiz To Course

Function Trigger: The instructor clicks "Assign Quiz" in the curriculum builder.

Function Description: Assigns an existing quiz to a specific chapter or course.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges and own the course.
  • Process
    • Assign Quiz Successfully:
      • Instructor selects a chapter and clicks "Add Item", then chooses "Quiz".
      • The system lists all available unassigned quizzes or allows selecting from the bank.
      • Instructor selects a quiz and confirms.
      • The system links the quiz to the course chapter.
      • The system updates the course curriculum layout.
  • Postconditions
    • The quiz is linked to the target course and is visible in the curriculum.
  • Error Handling
    • Quiz Already Assigned: The system prevents assigning a quiz that is already in another chapter of the same course.
  • Security Measures
    • Ownership Validation: The system ensures the instructor owns both the course and the quiz being assigned.
