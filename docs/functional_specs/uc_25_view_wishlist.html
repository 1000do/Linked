1. View Wishlist

Function Trigger: User accesses the wishlist page.

Function Description: Allows an authenticated User to view all courses they have saved in their personal wishlist. For each course, the system retrieves title, thumbnail, price, instructor name, average rating, enrollment count, and check if the User is currently enrolled in it.

Function Details

  • Preconditions
    • User must be logged in.
  • Process
    • View Wishlist Successfully:
      • User accesses the wishlist page.
      • The system checks if the User is authenticated.
      • The system extracts the user ID from the session claims.
      • The system queries the database to retrieve wishlist records associated with the User.
      • For each saved course, the system checks whether the User is already enrolled.
      • For each saved course, the system retrieves course statistics (average rating and total students).
      • The system renders and displays the wishlist page with the list of courses.
  • Postconditions
    • The User's wishlist courses are successfully displayed on the screen.
  • Error Handling
    • Session Invalid (Not Logged In): The system detects that the User is not authenticated and redirects the User to the Login page.
  • Security Measures
    • User must be logged in.
