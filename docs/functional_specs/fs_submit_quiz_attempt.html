4.20.2. Submit Quiz Attempt

Function Trigger: The user clicks the "Submit Quiz" button upon finishing the quiz.

Function Description: Submits the quiz attempt after completion.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must be logged in and have an active quiz attempt.
  • Process
    • Submit Quiz Attempt Successfully:
      • User completes the quiz questions and clicks "Submit Quiz".
      • The system asks for confirmation to submit.
      • User confirms the submission.
      • The system calculates the score based on the submitted answers.
      • The system updates the quiz attempt status to "Completed".
      • The system saves the results to the database.
      • The system redirects the user to the attempt detail page.
  • Alternative Flow (Auto-Submit on Time Up)
    • The system detects that the quiz time limit has expired.
    • The system automatically submits the user's current answers.
    • Back to step 4 of the normal process.
  • Postconditions
    • The quiz attempt is finalized, and the score is recorded.
  • Error Handling
    • Session Invalid: The system redirects the user to the Login page.
    • Attempt Already Submitted: The system prevents duplicate submission and displays an error.
  • Security Measures
    • Server-Side Time Validation: The system strictly validates the submission time against the server clock to prevent client-side time manipulation.
