UC-37. View Cart Items

Function Trigger: Actor clicks the "Shopping Cart" icon in the navigation bar header.

Function Description: Displays the list of courses added to the active user's shopping cart, calculates the pricing summary (subtotal, coupon discounts, final payable amount), and lists available checkout coupon options.

Screen Layout:
[Placeholder for UI mockups]

Function Details

  • Preconditions
    • Actor must be successfully authenticated.
  • Process
    • Access Cart:
      • Actor clicks the "Cart" icon on the navigation header.
      • The system verifies the active session and navigates the user to the Shopping Cart page.
    • Load Items:
      • The system displays the detailed list of courses (title, instructor, thumbnail, price) currently in the cart.
      • The system calculates and displays the Subtotal, applied Discount Amount (if any), and the final Payable Total.
      • Actor views the detailed shopping cart workspace and reviews their items.
  • Alternative Flow (Active Coupon Applied)
    • If a coupon code was previously applied to this order:
      • The system detects that a coupon code was previously applied to this order.
      • The system recalculates the discount and updates the final Payable Total.
  • Postconditions
    • The Actor views the detailed list of cart items and the finalized calculated total amount.
  • Error Handling
    • Session Invalid (Not Logged In): If the session has expired, the user is redirected to the login page before the cart can be viewed.
    • Empty Cart: If there are no courses currently in the cart, the system displays an empty notice: "Your cart is empty" with the description "Explore interesting courses and add them to your cart!" and an "Explore Courses" button.
    • Invalid Coupon in Cart: If any applied coupon is detected as invalid, expired, or doesn't meet minimum requirements, the system automatically removes that invalid coupon code from the user's coupon cookie map, recalculates the pricing summary, updates the cart display, and displays a specific error message indicating the coupon is invalid or expired.
  • Security Measures
    • Authentication Enforcement: Access to the cart details is restricted strictly to authenticated users.
