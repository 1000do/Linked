4.20.10. Delete Question

Function Trigger: The instructor clicks the "Delete" icon next to a question.

Function Description: Removes a question from the question bank.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges and own the question.
  • Process
    • Delete Question Successfully:
      • Instructor clicks "Delete" on a specific question.
      • The system asks for confirmation.
      • Instructor confirms.
      • The system removes the question or marks it as deleted (soft delete).
  • Postconditions
    • The question is no longer available in the question bank.
  • Error Handling
    • Constraint Error: If the question is assigned to a published quiz, the system blocks deletion and displays an error message.
