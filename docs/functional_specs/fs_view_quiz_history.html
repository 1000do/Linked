4.20.4. View Quiz History

Function Trigger: The user navigates to the quiz summary page.

Function Description: Displays the history of all completed quiz attempts.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must be logged in and enrolled in the course.
  • Process
    • View Quiz History Successfully:
      • User opens the quiz summary page.
      • The system queries all past attempts by the user for this specific quiz.
      • The system displays the list of attempts showing date, score, and status.
  • Postconditions
    • The attempt history list is presented to the user.
  • Error Handling
    • Session Invalid: The system redirects the user to the Login page.
