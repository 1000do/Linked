4.10.20. View Instructor's Course List

Function Trigger: Instructor navigates to the Instructor Dashboard.

Function Description: Allows an approved instructor to view their course portfolio, overall performance statistics, and manage individual courses from a centralized dashboard. The system ensures only fully approved instructors can access this view, redirecting others to appropriate application workflows.

Function Details

  • Preconditions
    • The actor must be authenticated.
    • The actor must have an approved instructor profile.
  • Process
    • View Dashboard Successfully:
      • Instructor accesses the Instructor Dashboard page.
      • The system validates the instructor's authentication and approval status.
      • The system retrieves the instructor's overall performance statistics (e.g., total enrollment, average rating, active courses, total revenue).
      • The system retrieves the instructor's paginated course list.
      • The system displays the Instructor Dashboard, rendering the performance statistics and the course list.
  • Postconditions
    • The system returns a paginated list of the instructor's courses along with their performance statistics.
  • Error Handling
    • .1 Unauthenticated Access: The system detects the actor is not authenticated. The system redirects the actor to the Login page.
    • .2 Instructor Application Pending or Rejected: The system detects the actor's instructor profile is not approved (e.g., pending or rejected). The system redirects the actor to the Application Status page.
    • .3 Instructor Profile Not Found: The system detects the actor does not have an instructor profile. The system redirects the actor to the Instructor Application page.
  • Security Measures
    • Instructor Dashboard Access Restriction: Only fully approved instructors are allowed to access the Instructor Dashboard; other states trigger redirects to application status or creation workflows.
    • Course Status Badging: The system visually distinguishes courses on the instructor dashboard using specific badges based on their current status (e.g., Published, Pending Review, Draft, Rejected, Archived, Removed, Permanently Locked).
    • Contextual Dashboard Actions: Course management actions on the instructor dashboard are determined dynamically based on course status; permanently locked or removed courses have severely restricted action sets.
    • Instructor Data Isolation: Instructors can only view statistics, revenue, and courses associated with their own account.
    • Universal Empty State Display: Whenever a list retrieval operation yields no records (either due to an empty database or active filter criteria), the system displays a user-friendly empty state message instead of a blank table or list container.
