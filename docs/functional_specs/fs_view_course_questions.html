4.20.7. View Course Questions

Function Trigger: The instructor filters the question bank by a specific course.

Function Description: Displays the list of questions associated with a specific course.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges and own the specific course.
  • Process
    • View Course Questions Successfully:
      • Instructor selects a specific course from the filter dropdown.
      • The system fetches questions linked to that course's quizzes.
      • The system displays the filtered list.
  • Postconditions
    • Only the course-specific questions are shown.
