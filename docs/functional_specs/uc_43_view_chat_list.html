UC-43. View Chat List

Function Trigger: Actor accesses the Chat page.

Function Description: Allows the Actor to view their list of active conversations, including partner avatar, partner name, last message snippet, timestamp, online status, and unread message count badge.

[Placeholder for UI mockups]

Function Details

  • Preconditions
    • Actor must be successfully authenticated.
    • If the Actor is an Instructor, they must be approved to access the instructor chat view.
  • Process
    • Access Page:
      • Actor accesses the Chat page.
      • The system sends a request to retrieve the list of active conversations.
    • Retrieve Conversations:
      • The system validates that the active user is authenticated.
      • The system queries the database for active chat rooms where the user is a participant.
    • Display List:
      • The system displays the list of conversations in the chat sidebar, showing partner details, last message content, last message timestamp, unread message count, and online status.
  • Alternative Flow (Filter by Unread)
    • If the Actor filters by unread messages:
      • Actor selects "Unread" from the message filter dropdown.
      • The system filters the chat sidebar list to show only conversations with an unread message count greater than zero.
  • Alternative Flow (Sort Conversations)
    • If the Actor sorts conversations:
      • Actor selects a sorting option (such as "Name (A-Z)" or "Unread First") from the sort dropdown.
      • The system re-orders the conversation cards in the sidebar accordingly.
  • Postconditions
    • The system displays the list of active conversations in the sidebar.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message and redirects the Actor to the login page.
  • Security Measures
    • Authentication Verification: The system validates that only authenticated users can fetch chat lists.
    • Instructor Status Validation: The system restricts instructor-specific chat views to approved instructors.
