4.20.13. Delete Quiz

Function Trigger: The instructor clicks "Delete" on a quiz.

Function Description: Deletes an existing quiz from the system.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges and own the quiz.
  • Process
    • Delete Quiz Successfully:
      • Instructor clicks "Delete" next to a quiz.
      • The system requests confirmation.
      • Instructor confirms.
      • The system unlinks the quiz from any curriculum and performs a soft delete.
  • Postconditions
    • The quiz is removed from the instructor's active list.
  • Error Handling
    • Active Attempt Error: Deletion may be prevented if students are currently taking the quiz.
