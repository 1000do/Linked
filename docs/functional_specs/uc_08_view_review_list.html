4.17.2. View Review List

Function Trigger: Actor accesses the Course Details page and scrolls to the reviews section.

Function Description: Displays the list of public course reviews, star ratings, and statistics, allowing sorting/filtering by star rating and pagination.

[Placeholder for UI mockups]

Function Details

  • Preconditions
    • Course details must be accessible.
  • Process
    • View Review List Successfully:
      • Actor accesses the Course Details page of a course.
      • The system retrieves the overall review statistics.
      • The system retrieves the first page of course reviews.
      • The system displays the average rating, star rating icons, total review count, and percentage distribution bars.
      • The system displays the list of reviews, showing each reviewer's avatar/initial, name, role badge, review date, star rating, course/lesson badge, and comment content.
      • The system displays the pagination controls containing "Prev" and "Next" buttons.
  • Postconditions
    • Rating statistics and reviews list are displayed successfully.
  • Error Handling
    • System Error Loading Reviews: The system displays an error message indicating that reviews could not be loaded. The flow terminates.
  • Security Measures
    • Anonymous Access: Anyone (including Guests) can view public course reviews and statistics.
    • Review Masking (Removed Reviews): If a review has been removed by the moderator (status "violating"), the content is replaced with a warning notification: "This review was removed by a moderator for violating community standards." and the rating stars are set to 0. If a review has been self-deleted by the user, it is omitted from the course details list.
    • Paging Size: The reviews list on the details page is paginated with a limit of 5 reviews per page.
    • General vs Lesson Badges: General course-level reviews are flagged with a "General review" badge, while lesson-level reviews (originally submitted in the player) are flagged with the corresponding lesson title.
