4.10.17. Edit Learning Materials

Function Trigger: Instructor clicks the "Save" button after editing video details, or clicks the "Save Changes" button after editing a resource file's details in the lesson editor.

Function Description: Allows an instructor to edit the title and description of an existing learning material (video or resource file). The system saves the updated details to the database. Modifying material details automatically reverts the course status to draft if it was previously published.

Function Details

  • Preconditions
    • User must be successfully authenticated as an approved instructor.
    • The material must exist and belong to a lesson in a course owned by the active instructor.
  • Process
    • Edit Material Details Successfully:
      • Instructor accesses the course edit page.
      • Instructor modifies the title or description of a video or resource file in the active lesson editor.
      • Instructor clicks the save button.
      • The system validates the instructor's rights.
      • The system validates the course status.
      • The system updates the learning material's details in the database.
      • The system updates the course status to draft and clears moderation feedback in the database.
      • The system invalidates cached course data.
      • The system displays a success confirmation message and updates the UI to reflect the updated details.
  • Postconditions
    • The material's title and description are updated in the database.
    • If the course was previously published, its status is reverted to draft and moderation feedback is cleared.
    • Stale cache data is cleared.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the user to the login page.
    • Invalid Instructor Rights: The system detects the instructor's rights are invalid. The system displays an error message. Back to step 1.
    • Invalid Course Status: The system detects the course status does not permit modifications. The system displays a restriction error message. Back to step 1.
    • Database Update Failed: The system fails to save the material changes. The system displays a save failure error message. Back to step 1.
  • Security Measures
    • Syllabus Change Reversion: Adding or removing lessons and materials, or editing their content, in a 'Published' course automatically changes the course status back to 'Draft' and clears any existing moderation feedback.
    • Permanently Discontinued Block: A course that is 'archived' and has 3 or more flags is considered permanently discontinued due to policy violations and cannot have its status changed or be edited.
    • Locked Out Instructor Rights: Instructors with an active lockout status are prohibited from submitting courses for review or changing course status.
