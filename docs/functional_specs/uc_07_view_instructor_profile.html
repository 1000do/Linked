4.10.6. View Instructor Profile

Function Trigger: Actor clicks on the instructor's name or profile link.

Function Description: Displays the public profile page of an instructor along with their list of created courses.

[Placeholder for UI mockups]

Function Details

  • Preconditions
    • The instructor's application must be approved.
  • Process
    • View Instructor Profile Successfully:
      • Actor clicks on an instructor's name or profile link.
      • The system retrieves the instructor's ID from the URL request.
      • The system retrieves the instructor's profile details (avatar, full name, professional title, expertise categories, biography, and social links).
      • The system retrieves the instructor's statistical information (total students, average rating).
      • The system counts the instructor's active courses.
      • The system retrieves the instructor's published courses.
      • The system displays the public profile page containing the instructor's avatar, full name, professional title, expertise categories, biography, social links ("LinkedIn", "YouTube", "Facebook"), and statistics (total students, active courses, average rating, total reviews).
      • The system displays a list of course card items for each published course, showing the course thumbnail, title, rating, price, and enrollment count.
  • Postconditions
    • The instructor's public profile page is rendered, displaying their biography, social links, statistical information, and a list of published courses.
  • Error Handling
    • Instructor Not Found or Not Approved: The system detects that the instructor does not exist or has not been approved. The system redirects the actor to the home page.
  • Security Measures
    • Course Visibility: Only courses with status "Published" are displayed on the public profile. Draft, pending, rejected, or archived courses are strictly excluded.
    • Profile Access: A public profile is only accessible if the instructor's account application status is "Approved".
    • Social Links: Social media links ("LinkedIn", "YouTube", "Facebook") are only rendered as clickable buttons/links if the instructor has provided the respective URLs.
