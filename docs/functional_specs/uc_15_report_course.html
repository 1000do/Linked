UC-15. Report a course

Function Trigger: User clicks the "Report this course" button on the course details page.

Function Description: Allows a user to report a course for violations (e.g., misleading content, copyright violation, spam, adult content). The system allows adding multiple reasons, validates the input, and submits the report to the server for moderation.

Function Details

  • Preconditions
    • User must be successfully authenticated.
    • User must be enrolled in the course.
    • User must not be the course owner or a manager.
    • The course must be in a reportable state (not pending, draft, or permanently locked).
  • Process
    • Report Course Successfully:
      • User accesses the course details page.
      • User clicks the "Report this course" button.
      • The system checks authentication.
      • The system displays the report course form.
      • User inputs the report details (reason and description).
      • User clicks the Submit Report button.
      • The system validates the report details.
      • The system submits the report data to the server.
      • The system saves the report in the database.
      • The system displays a success message regarding report submission.
  • Postconditions
    • The report is recorded in the database for moderation.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the user to the login page.
    • Database Save Failed: The system fails to save the report to the database. The system displays an error message regarding report submission failure. Back to step 2.
  • Security Measures
    • Universal Form Data Validation: All submitted form fields must pass strict data type, format, length, and value constraints (e.g., report descriptions must be at least 20 characters, and confidence thresholds must be decimals between 0.01 and 1.0) before the system processes the submission.
    • Unique Reasons: The user cannot select the same report reason more than once in a single submission.
