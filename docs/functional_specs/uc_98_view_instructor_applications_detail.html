UC-98. View Instructor Applications Detail

Function Trigger: Actor clicks the "View Info" link for a pending instructor application.

Function Description: Allows the Actor to inspect the detailed credentials of an instructor applicant, including their external LinkedIn, YouTube, and Facebook profiles, and submitted resume or portfolio documents.

[Placeholder for UI mockups]

Function Details

  • Preconditions
    • Actor must be successfully authenticated as a Staff member or Admin.
    • The instructor's application must exist in a pending status.
  • Process
    • Access Details:
      • Actor accesses the pending applications list page.
      • Actor clicks the "View Info" link.
    • Retrieve Details:
      • The system retrieves the detailed data of the selected instructor application.
    • Display Details:
      • The system displays the applicant's profile details, including their avatar, full name, email, professional title, expertise areas, social media links, and submitted documents.
      • The system displays the "Approve" and "Reject" buttons.
  • Alternative Flow (Cancel Viewing Details)
    • If the Actor cancels viewing details:
      • Actor clicks the "Cancel" button or the "Back to List" link.
      • The system redirects the Actor back to the pending applications list page.
  • Postconditions
    • The system displays the detailed credentials of the selected instructor applicant.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message and redirects the Actor to the login page.
    • Application Not Found: The system detects that the application ID is invalid or has been processed by another manager, displays an "Application not found" error message, and redirects the Actor back to the refreshed pending applications list.
  • Security Measures
    • Role-based Access Control: Access to the instructor application details is restricted strictly to authenticated users with Staff or Admin roles.
    • Data Validation: The server validates that the requested application ID exists and is currently in a pending state before returning the detailed data.
