4.10.3. Search Courses

Function Trigger: Actor enters a search term in the keyword input field on the explore courses page.

Function Description: Allows Guests and Users to search for courses by entering keywords that match titles, descriptions, and skills. The search is executed in real-time with a brief debounce delay, presenting matching results in the course catalog.

Function Details

  • Preconditions
    • None.
  • Process
    • Search Courses Successfully:
      • Actor accesses the explore courses page.
      • Actor enters a search term in the "Search Keyword" input field.
      • The system triggers the search query after a 350ms debounce delay.
      • The system retrieves the paginated list of courses matching the keyword.
      • The system displays the search results in the catalog grid.
  • Postconditions
    • Paginated list of courses matching the keyword search criteria is displayed successfully in the catalog grid.
  • Error Handling
    • None.
  • Security Measures
    • Debounced Query Execution: Real-time keyword filtering is debounced by 350ms to minimize API calls during continuous typing.
    • Keyword Match Scope: Search query checks against course titles, instructor names, short descriptions, and categories.
    • Published Status Only: Only courses with "published" status are returned in the search results.
