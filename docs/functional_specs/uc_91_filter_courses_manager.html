4.4.4. Filter Courses

Function Trigger: Actor selects a category from the "category" dropdown or a status from the "status" dropdown on the Course Moderation Page.

Function Description: Filters courses by categories or moderation status (Pending, Approved, Rejected).

Function Details

  • Preconditions
    • Actor must be authenticated.
    • Actor must have staff or admin role.
    • Actor must be on the Course Moderation Page.
  • Process
    • Filter Courses Successfully:
      • Actor selects a category from the "category" dropdown, or a status from the "status" dropdown.
      • The system retrieves the courses list matching the selected criteria and other active search keywords or sorting options.
      • The system updates the course moderation table and statistics cards.
      • The system displays the filtered course list.
  • Postconditions
    • The system displays the course moderation list filtered by the selected criteria.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the actor to the login page.
    • Unauthorized Access: The system detects that the actor does not have staff or admin roles. The system displays an access denied page or error message. The flow terminates.
    • API Call Failed: The system fails to retrieve the filtered course list or stats from the API. The system displays an error message regarding filtering failure. The system renders the Course Moderation page with an empty list. The flow terminates.
  • Security Measures
    • Exclusion of Drafts: Courses with status "Draft" are never retrieved or displayed in the moderation queues.
    • Category Filter: Matches the Category ID or Name against the course's category. If "All Categories" is selected, no category filtering is applied.
    • Independent Status Filters: The status filter in the Course Reports tab maintains its own independent state from the status filter in the Review Reports tab.
    • Preservation of Other Criteria: Filtering preserves active search keywords and sorting options.
