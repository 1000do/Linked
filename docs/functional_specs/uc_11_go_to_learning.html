4.10.8. Go To Learning

Function Trigger: Actor clicks the "Go to learning" button (or "Preview your course" button) on the course details page, or clicks the "Continue learning" button (or "Review" button) on the enrolled courses page.

Function Description: Opens the course player to watch video lectures, read text content, and view course slides.

Function Details

  • Preconditions
    • Actor must be authenticated.
    • Actor must be enrolled in the course, or be the owner (Instructor) of the course.
  • Process
    • Go To Learning Successfully:
      • Actor accesses the Course Details page or the "My Enrolled Courses" page.
      • Actor clicks the "Go to learning" button (or "Preview your course", "Continue learning", "Review" buttons).
      • The system checks the actor's session authentication.
      • The system validates the actor's enrollment status or course ownership.
      • The system retrieves the course structure, including lessons, lectures, videos, text content, and downloadable slides.
      • The system retrieves the actor's learning progress (completed materials list).
      • The system retrieves the rating average and total reviews count for each lesson section.
      • The system displays the course player view, showing the media area (with the first or last-viewed video lecture loaded), overall progress bar, and the curriculum list.
  • Postconditions
    • Course player page is displayed successfully, rendering the video player or document viewer, the list of lessons, and the overall course progress bar.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the actor to the login page.
    • Access Denied (Not Enrolled): The system detects the actor is not enrolled in the course and is not the course owner. The system redirects the actor to the error page.
  • Security Measures
    • Dynamic Button Update: For free courses, when an un-enrolled User clicks "Enroll Now", the page successfully processes the enrollment and dynamically updates the button to "Go to learning" without a full page reload.
