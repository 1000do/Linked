4.20.9. Edit Question

Function Trigger: The instructor clicks the "Edit" icon next to a question.

Function Description: Modifies the content and answers of an existing question.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges and own the question.
  • Process
    • Edit Question Successfully:
      • Instructor clicks "Edit" on a specific question.
      • The system opens the question form populated with existing data.
      • Instructor makes changes and clicks "Save".
      • The system saves the updated details to the database.
  • Postconditions
    • The question is updated.
  • Error Handling
    • In Use Warning: If the question is currently part of an active quiz with ongoing attempts, the system may block editing or warn the instructor.
