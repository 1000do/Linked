4.10.18. Remove Learning Materials

Function Trigger: Instructor clicks the "Remove" icon button next to a specific resource material, or clicks the "Replace Video" button for an existing active video.

Function Description: Allows an instructor to remove an existing learning material (such as a video or document) from a lesson. The system soft-deletes the material by moving the physical file to the cloud storage trash folder, updates the material's status to removed, and reverts the course status to draft if it was previously published. This use case also encompasses the automatic removal of an active video when a new video is uploaded to replace it.

Function Details

  • Preconditions
    • User must be successfully authenticated as an approved instructor.
    • The material must exist and belong to a course owned by the active instructor.
  • Process
    • Remove Material Manually Successfully:
      • Instructor accesses the course edit page.
      • Instructor selects a specific lesson from the curriculum list.
      • Instructor clicks the "Remove" icon button next to a specific resource material.
      • The system displays a removal confirmation dialog.
      • Instructor confirms the removal.
      • The system validates the instructor's rights.
      • The system validates the course status.
      • The system moves the physical file to the Cloud Storage Service's trash folder.
      • The system updates the material record's learning status to removed in the database.
      • The system updates the course status to draft and clears previous moderation feedback (if previously published).
      • The system invalidates the cached course data.
      • The system displays a success message.
  • Postconditions
    • The physical file is moved to the Cloud Storage Service's trash folder.
    • The material record in the database is updated with a removed learning status.
    • If the course was previously published, its status is reverted to draft and moderation feedback is cleared.
    • Stale cache data is cleared.
  • Error Handling
    • .1 Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the user to the login page.
    • .2 Material Not Found: The system halts execution and displays a material not found error message.
    • Invalid Instructor Rights: The system detects the instructor is unauthorized to remove this material or has an active lockout status. The system displays an error message regarding invalid instructor rights. Back to step 3.
    • Invalid Course Status: The system detects the course is pending review or is permanently archived due to policy violations. The system displays an error message regarding invalid course status. Back to step 3.
    • Database Save Failed: The system fails to save the material changes to the database. The system displays a database transaction failure message. Back to step 3.
  • Security Measures
    • Locked Out Instructor Rights: Instructors with an active lockout status are prohibited from submitting courses for review or changing course status.
    • Pending Review Modification Lock: Learning materials and lessons cannot be added, edited, or removed if the course is currently in 'Pending' review status.
    • Permanently Discontinued Block: A course that is 'archived' and has 3 or more flags is considered permanently discontinued due to policy violations and cannot have its status changed or be edited.
    • Material Soft Deletion: Removing a material does not permanently delete the file; the physical file is moved to the Cloud Storage Service's trash folder and its database status is updated to 'Removed'.
    • Video Swap Constraint: Restoring a video material to a lesson that already contains an active video will automatically move the current active video to the trash bin, replacing it with the restored video.
    • Syllabus Change Reversion: Adding or removing lessons and materials, or editing their content, in a 'Published' course automatically changes the course status back to 'Draft' and clears any existing moderation feedback.
