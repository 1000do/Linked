4.17.1. Add Personal Reviews

Function Trigger: Actor submits a rating and comment for a course (or lesson) they are enrolled in or own.

Function Description: Allows learners to submit reviews and star ratings for courses (or lessons) they are enrolled in, enforcing learning progress checks.

[Placeholder for UI mockups]

Function Details

  • Preconditions
    • Actor must be authenticated.
    • Actor must be enrolled in the course, or be the owner (Instructor) of the course.
  • Process
    • Submit Course Review from Course Details Page Successfully:
      • Actor accesses the Course Details page of a course.
      • The system checks the actor's session authentication.
      • The system checks if the actor is the owner of the course.
      • The system retrieves the course structure statistics.
      • The system retrieves the actor's completed materials count.
      • The system validates that the actor's learning progress is 100%.
      • The system displays the write review form.
      • Actor clicks the desired "star" icon to select a rating and enters comments in the review comment input.
      • Actor clicks the "Submit Review" button.
      • The system validates that a star rating is selected.
      • The system validates that the comment content is not empty.
      • The system checks if the actor's account has an active review posting restriction.
      • The system saves the course review in the database.
      • The system displays a success message.
      • The system dynamically updates and reloads the reviews list and rating statistics.
      • The system sends a new review notification to the course instructor.
      • Actor accesses the Course Player page (Go to learning).
      • The system checks the actor's session authentication.
      • The system checks if the actor is the owner of the course.
      • The system retrieves the course structure statistics.
      • The system retrieves the actor's completed materials count.
      • The system validates that the actor has completed at least 1 lesson.
      • The system displays the write review form in the Reviews tab.
      • Actor clicks the desired "star" icon to select a rating and enters comments in the review comment input.
      • Actor clicks the "Submit Review" button.
      • The system validates that a star rating is selected.
      • The system validates that the comment content is not empty.
      • The system checks if the actor's account has an active review posting restriction.
      • The system saves the lesson review in the database.
      • The system displays a success message.
      • The system dynamically updates and reloads the reviews list, overall rating statistics, and lesson rating statistics.
      • The system sends a new review notification to the course instructor.
  • Postconditions
    • The review record (either Course Review or Lesson Review) is successfully created and saved in the database.
    • If the reviewer is not the course owner, a notification is sent to the course instructor.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the actor to the login page.
    • Restricted Account: The system detects that the actor's account has an active review posting restriction. The system displays an error message indicating that their account is restricted from posting comments or reviews and states the restriction end time. The flow terminates.
  • Security Measures
    • Completion Check for Course Review: Non-owner students must have 100% course progress (completed all materials) to write a review on the Course Details page.
    • Minimal Progress for Lesson Review: Non-owner students must have completed at least 1 lesson/material to write a review on the Course Player page.
    • Instructor Bypass: Course owners/instructors can access the Course Player ("Learn" page) for preview, and their progress is treated as 100% completed virtual progress without writing completions to the database.
    • Rating Restrictions: Star ratings must be between 1 and 5 stars (inclusive).
    • Account Restrictions (Lockout): Users with active comment/review restrictions are blocked from submitting new reviews. The system checks this validation at submission time.
    • Instructor Notification: Upon successful submission, a notification is sent to the course instructor with a link pointing directly to the review card.
