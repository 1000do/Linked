4.20.8. Create Question

Function Trigger: The instructor clicks the "Add New Question" button.

Function Description: Creates a new multiple-choice question in the bank.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges.
  • Process
    • Create Question Successfully:
      • Instructor clicks "Add New Question".
      • Instructor enters the question text, multiple choice options, and marks the correct answer(s).
      • Instructor clicks "Save".
      • The system validates the inputs (e.g., ensuring at least one correct answer).
      • The system saves the question to the database.
      • The system updates the question bank list.
  • Postconditions
    • The new question is available in the question bank.
  • Error Handling
    • Validation Error: Missing question text or no correct answer marked will trigger a validation alert.
