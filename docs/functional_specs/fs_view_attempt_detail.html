4.20.3. View Attempt Detail

Function Trigger: The user clicks on a completed attempt from their quiz history.

Function Description: Displays the detailed result of a specific quiz attempt.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must be logged in.
    • The quiz attempt must be completed and belong to the user.
  • Process
    • View Attempt Detail Successfully:
      • User accesses the quiz attempt detail page.
      • The system retrieves the attempt details, including score, time taken, and submitted answers.
      • The system checks the quiz settings to determine if correct answers should be displayed.
      • The system renders the detailed report.
  • Postconditions
    • The detailed result of the specific quiz attempt is displayed to the user.
  • Error Handling
    • Attempt Not Found: The system displays a "Not Found" error if the attempt ID is invalid.
    • Unauthorized Access: If a user tries to view another user's attempt, they are blocked.
