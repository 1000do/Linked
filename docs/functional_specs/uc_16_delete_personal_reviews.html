4.17.3. Delete Personal Reviews

Function Trigger: Actor clicks the "Delete" button on a review they previously submitted.

Function Description: Allows users to delete the course reviews they have previously submitted.

[Placeholder for UI mockups]

Function Details

  • Preconditions
    • Actor must be authenticated.
    • Actor must be the author of the review.
  • Process
    • Delete Personal Review Successfully:
      • Actor accesses the Course Details page (or Course Player page) where their review is displayed.
      • Actor clicks the "Delete" button on their own review card.
      • The system displays a confirmation popup dialog.
      • Actor clicks the "Yes, delete it" button.
      • The system checks the actor's session authentication.
      • The system checks if the review is currently under moderation review.
      • The system checks if the actor is the owner of the review.
      • The system soft-deletes the review in the database.
      • The system displays a success notification indicating the review has been deleted.
      • The system dynamically reloads the reviews list and updates rating statistics.
  • Postconditions
    • The review is soft-deleted in the database.
    • The reviews list and rating statistics are updated dynamically.
  • Error Handling
    • Session Invalid (Not Logged In): The system displays an authentication error message. The system redirects the actor to the login page.
    • Review Under Moderation: The system detects that the review has an active pending moderation report. The system displays an alert message indicating that the review is currently under moderation review and cannot be deleted. The flow terminates.
    • Unauthorized Access: The system detects that the actor is not the author of the review. The system displays an error message indicating that they can only delete their own reviews. The flow terminates.
  • Security Measures
    • Self-Ownership Only: An actor can only delete reviews they have authored.
    • Moderation Block: If a review has an active pending moderation report, deletion is blocked until the moderation report is resolved.
    • Soft Delete: Deletion is implemented as a soft-delete (setting `IsRemoved = true` and updating status to "removed" in the database). It does not permanently delete the record.
    • Rating Statistics Recalculation: Upon successful deletion, the overall rating average and counts are dynamically recalculated to exclude the deleted review.
