4.20.14. Edit Quiz Settings

Function Trigger: The instructor clicks "Settings" on an existing quiz.

Function Description: Modifies quiz configurations such as time limits and passing scores.

Screen Layout:
[image_placeholder]

Function Details

  • Preconditions
    • The user must have Instructor privileges and own the quiz.
  • Process
    • Edit Settings Successfully:
      • Instructor accesses the quiz settings page.
      • Instructor updates configurations (e.g., passing score, attempt limits, time limit).
      • Instructor clicks "Save Changes".
      • The system updates the quiz properties in the database.
  • Postconditions
    • The quiz rules are updated and will apply to future attempts.
